# discord-userdoccers-tracker

Tracking missing routes on [Discord Userdoccers](https://discord.food).

## Files & Folders

- `logs.md` – Human-readable results showing which routes are covered and which are missing.  
- `logs.json` – JSON format of the results with `covered` and `missing` arrays.  

- `./data/` – Folder containing scraped data from Userdoccers.  
  - `merged.mdx` – All MDX files from Userdoccers combined into one file.  
  - `routes.json` – JSON file containing every route in a clean format.
